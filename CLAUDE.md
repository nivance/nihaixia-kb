# 倪海厦中医知识库

## 目录结构

本知识库采用双层架构：

- **raw/** — 原始文档（只读参考），保留原文完整性
- **wiki/** — 结构化知识页面，从 raw 提取，支持双向链接

## 查询规则

1. 默认从 `wiki/` 目录查询中医相关资料
2. 搜索维基索引 `wiki/INDEX.md` → 读取相关页面 → 综合回答
3. 如需查看原始文档，参考 `raw/` 下对应目录
4. 回答时引用具体文件路径和元数据

## Wiki 页面类型

| 类型 | 说明 | 位置 |
|------|------|------|
| 摘要 | raw 的摘要、关键点 | 各来源下的摘要/目录 |
| 症状 | 症状及望闻问切信息 | 各来源下的症状/目录 |
| 病机 | 推断的病生理状态 | 各来源下的病机/目录 |
| 理论 | 中医理论片段 | 各来源下的理论/目录 |
| 治法 | 治疗方法 | 各来源下的治法/目录 |

## 链接约定

- `[[页面]]` — 内部链接
- `[[页面|显示文本]]` — 自定义文本链接

## 工作流

### 摄取流程（人纪）

1. 读取源文档 → 提取症状、病机、理论、治法
2. 创建摘要页面
3. 更新关联页面（症状、病机、理论、治法）
4. 更新索引和日志
