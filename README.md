# 倪海厦中医知识库

倪海厦先生中医著作的统一知识库，采用双层架构。

## 目录结构

```
nihaixia-kb/
├── raw/             # 原始文档（只读参考）
│   ├── 人纪/       # 人纪五部经典
│   ├── 天纪/       # 天纪系列
│   ├── 地纪/       # 地纪
│   ├── 医案/       # 临床医案
│   ├── 方剂/       # 方剂知识
│   ├── 药物/       # 药物知识
│   ├── 穴位/       # 穴位知识
│   ├── 讲座/       # 讲座与演讲
│   ├── 专题/       # 专题文章
│   └── 其他/       # 相学、易筋经、闭门课
│
├── wiki/            # 结构化知识页面
│   ├── 人纪/       # 从人纪提取的知识
│   ├── 天纪/       # 从天纪提取的知识
│   ├── 地纪/       # 从地纪提取的知识
│   ├── 医案/       # 从医案提取的知识
│   └── 其他/       # 讲座、专题
│
└── scripts/         # 构建和校验脚本
```

## 说明
* 知识库所有资料均来自于github后重新做了整理
