## 概述

倪海厦在医案中广泛运用压痛点诊断法，将体表特定位置的压痛与内脏病变相关联。其核心原理是"痛皆来自压力"——有压痛代表该脏腑有实邪（结石、痰湿、瘀血等），通过按压脊柱、耳穴、四肢特定穴位，可快速定位内脏病变、判断病性虚实、鉴别病种、评估疗效。

---

## 一、脊柱压痛点（督脉诊断）

脊柱压痛是倪师最核心的压痛诊断体系，通过按压各椎体棘突下的压痛来判断脏腑病变。

| 椎体 | 对应病变 | 案例 |
|------|---------|------|
| **T3（第三胸椎）** | 肺家问题 | 090 O A：T3压痛 + 耳肺点压痛 → 肺家问题 |
| **T5（第五胸椎）** | 心包痰湿 / 心脏功能不足 | 010 B J：T5压痛 + 耳穴心区压痛 + 舌苔黄湿 → 心包痰湿盛 |
| **T5-T6之间** | 介于红斑性狼疮与血癌之间 | 221 L J：T5、T6压痛，T6减轻 |
| **T6（第六胸椎）** | 血癌（白血病） | Liu Qun：T6有压痛 → 白血病病根仍在 |

### T5压痛三联征

T5椎压痛 + **耳穴心区压痛** + **舌苔黄湿** → 心包痰湿盛。此三联征是倪师诊断心包痰湿的核心依据，治疗用乌梅丸（HT-40）苦温化湿。

### T5/T6压痛鉴别

- **T5压痛 + T6无压痛** → 心脏有问题但无血癌（096 P T）
- **T5+T6均有压痛** → 介于红斑性狼疮与血癌之间（221 L J）
- **T6压痛** → 白血病病根仍在（Liu Qun、132 W JM）
- **T6无压痛** → 白血病已愈（Logan Jurgen、132 W JM治疗后）

## 二、耳穴压痛点（耳诊）

倪师常用耳穴压痛作为辅助诊断，与脊柱压痛互参。

| 耳穴位置 | 对应病变 | 案例 |
|---------|---------|------|
| **心区** | 心包痰湿、心气不伸 | 010 B J、012 C CY、103 R D、154 B E、212 L HJ |
| **肝区** | 肝实（胆结石/胆囊问题） | 025 D R：耳肝区压痛 → 胆结石大柴胡汤证 |
| **胆石点** | 胆结石/胆囊阻塞 | 025 D R、103 R D、Chou Bi yuen、Wang Mary、Wright Nancy |
| **肾点** | 肾结石 | 238 T L、128 U B、095 P R、226 P J、140 Y L |
| **肺点** | 肺家问题 | 090 O A（肺点压痛）、207 L J（左下肺点压痛 → 肺家阴实） |
| **脾与胆点** | 脾胃/胆问题 | 127 T S |
| **子宫卵巢区** | 妇科问题 | 189 H C（无压痛 → 良性肌瘤） |
| **足跟压痛点** | 足跟痛 | 017 C HF：耳穴足跟压痛点埋耳针 |

### 耳诊鉴别要点

- **右肋下痛 + 耳肝区有压痛** → 实证（胆结石等）
- **右肋下痛 + 耳肝区无压痛** → 非实证（028 E G，阳明经热）
- **肾区痛 + 耳肾区压痛 + 肾石点压痛 + 小便淡黄无血尿** → 肾结石
- **肾区痛 + 血尿** → 肾脏癌（095 P R解说）

## 五、压痛点诊断的核心原则

### 1. 压痛 = 压力 = 实

倪师认为"痛都是来自压力"，有压痛代表该脏腑有实邪阻塞。如025 D R解说："此人是因燥屎塞住，胆管亦堵住才造成压力，有压力才会痛。"

### 2. 压痛鉴别虚实

同一部位的疼痛，结合压痛的有无可鉴别虚实：
- 右肋下痛 + 耳肝区压痛 → 实证（胆结石）
- 右肋下痛 + 耳肝区无压痛 → 非实证（028 E G）

### 3. 压痛鉴别良恶性

- **肾结石**：肾区痛 + 耳肾区压痛 + 肾石点压痛 + 小便淡黄无血尿
- **肾脏癌**：肾区痛 + 血尿（095 P R解说）
- **胆结石**：胁痛 + 胆石点压痛 + 耳胆点压痛
- **胰脏癌**：胰区痛 + 耳胰区压痛（035 G Y）

### 4. 压痛判断疗效

治疗后压痛减轻或消失 → 病情好转：
- 096 P T："T5已无任何压痛" → 心功能回复，血癌即无
- 221 L J："T5、T6的压痛已好转许多"
- 132 W JM："第六椎下无压痛，可改用胶囊"

### 5. 个体差异

有些人对痛不敏感，体表肾石点无压痛但耳穴肾点有反应（238 T L解说："有些人对痛不敏感。故此人虽有肾结石，但在肾石点并没有反应出来。但是在耳朵的肾点有反应。"）

## 六、与压痛点相关的常用方剂

| 方剂 | 对应压痛 | 病机 |
|-----|---------|------|
| **乌梅丸（HT-40）** | T5椎压痛 + 耳穴心区压痛 + 舌苔黄湿 | 心包痰湿盛 |
| **大柴胡汤** | 耳肝区压痛 + 胁痛 + 便秘 | 胆结石/胆道阻塞 |
| **HT-14（胆结石方）** | 胆石点压痛 | 胆结石 |
| **HT-26（肾结石方）** | 肾石点压痛 | 肾结石 |

---

## 七、与压痛点相关的针灸治疗

倪师常在压痛点处埋耳针或直接针刺：
- **耳穴压痛点** → 埋耳针（保持持续刺激），如心点、胆石点、肾石点、足跟点
- **胆石点/肾石点** → 直接针刺或埋针
- **T5/T6压痛** → 针心俞、督俞
- **耳穴颈点压痛** → 埋耳针解颈痛（104 R G）

---

## 八、总结

倪海厦的压痛点诊断体系是一个**"体表-内脏"对应系统**，通过脊柱（督脉）、耳穴、四肢特定穴位的压痛，快速定位内脏病变，并以此判断病性（虚实）、鉴别病种、评估疗效。其中**T5-T6脊柱压痛**和**耳穴诊断**是最核心的两大支柱，两者互参可大幅提高诊断准确性。
