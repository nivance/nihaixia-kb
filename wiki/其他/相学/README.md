# 倪海厦相学注解

---


### 1. 项目简介

倪海厦老师于《天纪》中讲授面相真诠，但因文字篇章，视频时长限制，老师的面相知识未能成系统体系，此诚为易学界之一大憾事。倪师生前倡导“五术兼修”（山、医、命、相、 卜）并身体力行，于命理则特为强调“命相同参”，然相较于已有成法的紫微斗数，老师的面相知识则碎片分散，后学之人难免望洋兴叹。

纵观当下，坊间总结虽多，但多是知识堆砌，少有系统总结，更缺批注与实证，后人研习则入路无门，叹学海无涯。笔者不才，承蒙家师指点，斗胆对倪师面相知识进行系统梳理与注释，辅以例证，但水平有限，错漏难免，故特此开源。愿众学友共研共磨，各成一家之注，共续倪师学术薪火。
